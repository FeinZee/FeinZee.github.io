this到底指向什么？

2019-07-08

        先认识了C++的this，再认识js的this就什么也不认识了，所以想在这里捋一下this指向什么。

        总括地说，this指向它所属的对象。但要注意它的使用位置。

  • 单独的情况下，this 指的是全局对象。
  • 在函数中，this 指的是函数被调用时的所属。
  • 在函数中（严格模式下），this 是 undefined
  • 在事件中，this 指的是接收事件的元素。

单独情况

如果在js文件中，直接看this：

console.log(this);

我们得到的结果是：

[object window];

因为此时this就属于window。

在函数中

this属于某个函数，要看函数是在哪里被调用的

情况一：

function f(){
    return this;
}
console.log(f());

我们得到的结果是：

[object window];

因为此时this所在的函数就是在全局被调用的。

情况二：

function f(){
    return this;
}
let obj = {
    name: 'Peter',
    func: f
}
console.log(obj.func());

我们得到的结果是：

[object Object];

因为此时this所属的函数f()是作为一个对象方法被调用的，所以这里this就指向对象。

在函数中(strict 模式)

如果是直接全局下调用函数：

情况一：

'use strict';
function f(){
    return this;
}
console.log(f());

我们得到的结果是：

undefined;

如果是类似上面的情况二，结果不变，依然指向对象。
